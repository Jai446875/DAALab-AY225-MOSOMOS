1. How to Run the Code
If the GUI does not appear after clicking run:

Locate the Run button at the top right of your editor.

[!WARNING] Note: If the window still doesn't pop up, try double-clicking the "Run" button or the "Run Java" option to ensure the process initializes correctly.

2. Loading and Sorting Data
Once the GUI window appears:

Click the "Load CSV File" button. This will open a file explorer.

Navigate to your dataset (e.g., generated_data.csv) and select it.

Click the "Run Sort" button to begin the process.

3. Configuring the Parameters
Directly below the "Load CSV File" button, you will find the configuration row:

Algorithm: Choose between Bubble Sort, Insertion Sort, or Merge Sort.

Sort By: Select which column (ID, First Name, or Last Name) you want to use as the sorting key.

N Rows: Type the number of records you want to sort (e.g., 1000, 10000, or 100000).
